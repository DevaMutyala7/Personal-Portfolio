PicSome

An Online Photo Store Web App

Description

This is a simple react App which allows you to buy pictures and use for your work. This is a Responsive web Application where it is implemented using React-Router, useState, useEffect, custom Hooks, React Context, Lazy Loading.

Data for this application is coming from external API and I have used Firebase for Authentication.
